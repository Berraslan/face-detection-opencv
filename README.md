# Real-Time Face & Eye Detection (OpenCV)

## 📌 Overview
This project is a real-time computer vision application that detects human faces and eyes using OpenCV's Haar Cascade classifiers. It processes live webcam input and highlights detected facial regions with bounding boxes.

---

## ⚙️ Technologies Used
- Python
- OpenCV
- Haar Cascade Classifiers

---

## 🚀 Features
- Real-time face detection via webcam
- Eye detection within detected face regions
- Bounding box visualization for faces and eyes
- Lightweight and fast classical computer vision approach

---

## 🧠 What I Learned
- Basics of computer vision pipelines
- Image preprocessing (grayscale conversion)
- Object detection using Haar Cascades
- Real-time video stream processing with OpenCV
- Performance tuning using parameters like scaleFactor and minNeighbors
